# HiDrive-SOC
Battery Hardware-in-the-Loop Validation of Electric Vehicle Powertrain Energy Management for State-of-charge Estimation under Dynamic Driving Conditions.

(1) Please place the CSV files in the correct directory so that HiDrive-SOC.py can run properly.
python HiDrive-SOC.py
(2) SerialComm.py contains the main hardware control logic, but running it requires the corresponding hardware setup.
